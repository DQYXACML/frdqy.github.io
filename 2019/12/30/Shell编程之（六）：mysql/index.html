mysql命令参数

命令 含义
-u 用户名
-p 用户密码
-h 服务器ip地址
-D 链接的数数据库
-N 不输出列信息
-B 使用tab键代替默认交互分隔符
-e 执行sql语句
-E 垂直输出
-H 以HTML格式输出
-X 以XML格式输出

注意

-B用来不显示周围一圈分隔符时建议放在选项最后，在-e之前。

命令参数

命令 含义
-u 用户名
-p 密码
-h 服务器IP地址
-d 等价于–no-data，只导出表结构
-t 等价于–no-create-info，只导出数据，不导出建表语句
-A 等价于–all-databases
-B 等价于–databases，导出一个或多个数据库
