系统安装自动化

发表于 2020-01-06 分类于 Linux

总览

电脑启动时，首先bootloader加载kernel，然后为了防止kernel不能识别硬盘，kernel首先使用ramdisk在内存中模拟一个根文件系统，这就不需要读取真正的磁盘，而且这个模拟文件系统有操作系统安装时需要的各种程序。之后，启动anaconda程序来完成系统安装，这个程序处在待安装系统的某个程序包下即可装载运行。
